# Security Policy

## Supported Versions

The following versions of this project are currently being supported with security updates:

| Version  | Supported          |
| -------- | ------------------ |
| >= 1.0.0 | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please report it by opening a GitHub Issue. We will investigate and address the issue as soon as possible.
